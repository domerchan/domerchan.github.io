Modelado con AnyLogic

AnyLogic es una herramienta de modelado de simulación multimétodo desarrollada por The AnyLogic Company. Está escrita en lenguaje Java SE, y es compatible con sistemas operativos Windows, macOS y Linux.

Requerimientos de Hardware
  • 1.5GB de espacio libre en el disco
  • 4-8GB de memoria RAM
  • Un procesador moderno dualcore
  • De preferencia un mouse
Requerimientos de Software
  • Para correr AnyLogic es necesario Java SE 11 o más

Desarrollar un modelo de simulación dentro de un software de simulación que contemple las siguientes funcionalidades:

  • Tener en cuenta un solo recinto electoral
  • Tomar los resultados de la regresión para la vacuna
  • Un promedio del 80% de personas realizarán el proceso de vacunación
  • Dentro del proceso del 5 al 10% no podrá vacunarse
  • Las personas solo tienen un reecinto electroral para realizar el proceso
  • Las personas realizan la primera vacuna, y después de 30 días la segunda
  • Las personas hacen fila para recibir la vacuna
  • La vacunación se realiza en un tiempo aleatorio entre 5 a 10 minutos
  • Debe eesperar 20 minutos dentro del establecimiento para verificar que no tenga problemas de salud
  • La persona recibe su certificado de vacunación y la fecha de la próxima vacuna entre 2 a 3 minutos
  • La persona sale del recinto electoral
  • Regresa para la próxima fecha y se repite el ciclo

Modelo

Simulación del Proceso de Vacunación ante COVID

Espacio

Primero se genera el espacio de simulación donde van a interactuar los agentes (dato que fluye por todo el proceso de simulación) con los recursos (dato que se reserva para un agente hasta que sea liberado). Para este modelo se han definido cuatro áreas. La zona de llegada, donde los pacientes esperan a que un médico esté libre para vacunarlo. La segunda es la zona de vacunación donde el paciente (agente) interactúa con el médico (recurso), la zona de espera, y la zona de carnetización.

Diagrama

Se crea un diagrama con la lógica del comportamiento de los agentes con los recursos. Para este caso se crean dos fuentes de agentes, pacientes para la primera y segunda dosis. Luego se define un condicional para que solo el 95% de los pacientes sea vacunado. Luego recorre las diferentes salas, respetando los rangos de tiempos establecidos. Después de que el paciente se vacuna entre 5-10 minutos, espera durante 20 minutos, y recibe su carnet en 2-3 minutos, se retira del recinto.
Parámetros

Se establecen parámetros para controlar la simulación. El tiempo de llegada de los pacientes, horarios de atención, rangos de tiempo de espera para cada zona, cantidad de recursos, tamaño de la población, etc. De estos parámetros dependerá el comportamiento de la simulación.
Gráficas

Se generan gráficas sobre el avance de los agentes para comprender de mejor manera la evolución de la simulación. Se generaros tres gráficas. La primera muestra el tiempo de espera total de un paciente desde que llega al recinto hasta que sale vacunado. Luego se muestra el tiempo de espera más concurrente y el promedio del tiempo de espera de todos los pacientes atendidos hasta la fecha. Finalmente se muestra una comparación de pacietes vacunados vs. no vacunados.

Conclusiones

AnyLogic es una herramienta de simulacón privada, que cuenta con una versión gratuita para aprendizaje o uso personal. Esta herramienta permite realizar un modelo de simulación de forma rápida e intuitiva por medio de su interfaz amigable basada en Eclipse IDE, así que si lo has usado el aprendizaje de esta plataforma será mucho más fácil. Ademas de generar modelos, AnyLogic permite generar un espacio gráfico para visualizar la evolución de la simulación en el tiempo, permitiendo comprender de forma más clara cuál es el comportamiento de las variables, agentes y recursos establecidos. Esta herramienta te permite generar desde modelos simples sin necesidad de tener muchos conocimientos en programación ya que la configuración se realiza por medio de íconos drag and drop, hasta modelos complejos donde se puede utilizar tu propio código escrito en Java (o Python con la ayuda de librerías).
